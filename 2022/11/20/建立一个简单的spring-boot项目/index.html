可能出现的问题

问题：报一些gradel相关的错误

原因：idea版本和gradel版本不兼容

解决方案：修改gradel.wrapper路径的gradle-wrapper.properties文件下的distributionUrl，寻找idea版本对应的gradel版本。

可能出现的问题

报错：ERROR org.springframework.boot.SpringApplication - Application run failed

原因：文件编码格式不对，需要改成utf-8

解决方案：setting->project encoding->修改为utf-8
